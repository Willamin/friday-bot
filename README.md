# friday-bot

Cool bot to post Friday videos to Slack

## Usage

```bash
# Configure environment; edit where appropriate
cp .env.example .env

# Build app
shards build

# Run app
bin/friday-bot-crystal
```

## Development

```bash
# Lint code
crystal tool format
```
